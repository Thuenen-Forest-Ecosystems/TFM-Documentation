# Data Structure

The data can be found in the inventory_archive schema. The inventory_archive schema contains the following tables:
```
- inventory_archive
    - cluster 
        - plot
            - edges
            - deadwood
            - regeneration
            - structure_lt4m
            - subplot_position
            - tree
```
